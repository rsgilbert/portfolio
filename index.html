Projects I am proud of

AVSI Leave Management System (LMS)

LMS is a web application I developed for AVSI Uganda. It is used by their employees to apply for leave and also fill in their timesheets.

The system also provides HR personnel with reports for analysing leave days taken, timesheet submission among others.

Technologies used:

  • HTML
  • CSS
  • Node.js

Sub Grant Management System (SGMS)

I developed a web application called SGMS for Mildmay Uganda. It is used by Mildmay's sub partners to request for funds and to also fill in accountabilities.

SGMS also has an audit module where the Audit Manager can schedule audits, track audit findings etc.

Technologies used:

  • React.js
  • SOAP

Electronic Payment System (EPS)

EPS is a web application I developed for Mildmay Uganda. It is used by Mildmay's officers to process salaries for their staff.

The facility officers upload reports into the system informing on which staff worked and should be paid.

The application supports an approval process from the facility staff upto the HR manager.

Technologies used:

  • React.js
  • Azure Blob Storage
